Document Object Model (DOM)

  • Created by the Browser -- when HTML document loads inside it
  • 'document' Object -- contains many properties and methods to interact with the document like add/remove content
  • Console > type: document > press: Enter > shows: #document -- visual representation of the html document
  • >> the html document is modeled by the document object which is created by the browser
  • inside the JS, the code gives us access to that document object and its used to interact with HTML pages using the different properties and methods
  • DOM also describes how the document looks in a visual way too
  • DOM sees HTML page as a heirarchical tree of nodes -- each element (html (root node), head, title (text node), body, h1 (text node), div, p (text node)) is considered a node in the DOM
  • DOM is used to reach into tree of nodes and we can get a reference to a particular node (ex: h1 node - refernce to that h1 we can use some different methods on that reference to change its content or change its style or remove it or change something else in it)
