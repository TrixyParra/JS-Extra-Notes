Functions and Methods

  • Functions - creates blocks of codes that can be reusable at any point in the code
    1. Is an Object Data Type
    2. Defines a block of code that we can run/call and execute whenever we want
    3. Create a function "function name()" that can be run multiple times
    4. Values can be in functions (ex: coordinates to actual address)
    5. Defined on it's own not an Object or Data Type
  • Methods - is a function that does something
    1. Difference is the way they are invoked and where they are defined
    2. Method is invoked/called using dot notation
    3. Associated with an Object or a Data Type like a String
    4. Defined on an Object
